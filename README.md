# tourr-tests

GSoC test documentation: Projection Pursuit Indices based on Scagnostics in the **tourr** package.

This repository contains three test levels demonstrating the implementation and application of the **skinny projection pursuit index**:

- **Easy**: Basic index implementation
- **Medium**: Guided tour vignette on real data
- **Hard**: End-to-end financial analysis pipeline
